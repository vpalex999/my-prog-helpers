Базовые команды GIT

Узнать текущее состояние

git status

Посмотреть изменения в непроиндексированных файлах

git diff

Посмотреть изменения в проиндексированных файлах

git diff --staged "имя проиндексированного файла"

Как откатить закомиченное изменение

git checkout "имя файла"

Если случайно удалили закомиченый файл

git checkout "удалённый файл"

Посмотреть историю комитов

git log

git log --oneline

git log -1 показывает только последний коммит

Удалить файл из коммита

git checkout "хеш коммита" "имя файла"

удалённый файл появится в: Changes to be comitted

Как убрать файл из индекса

git reset HEAD "имя файла"

Как закоммитить

git commit -m 'some text'

Как посмотреть коммит

git show 'хэш коммита'

Как исправить сообщение в последнем коммите

git commit --amend -m "new text"

Как убрать файл из последнего коммита

git rm "filename" удаляет файл на fs

git rm --cashed 'filename' только исключить из коммита без удаления на fs

git commit --amend --no-edit
